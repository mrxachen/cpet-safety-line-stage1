# 多变量异常评分报告（Method 2 / Phase G）

## 方法概述

基于参考人群的 Mahalanobis 距离 D² 捕获多变量 CPET 空间中的异常模式，
克服单变量 %pred 无法处理变量间协方差的局限。

D² ~ χ²(k) 分布（k = 变量数），提供概率解释。
