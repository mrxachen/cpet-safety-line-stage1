# 一、中国 CPET 规范与安全背景（引言必引）

## 1. 中国专家共识（2022）
**心肺运动试验临床规范应用中国专家共识**  
用途：
- 引言说明 CPET 在鉴别诊断、预后评估、运动处方和安全规范中的地位
- 方法部分说明中国本土临床语境

建议放置：
- 引言第一段
- 方法中的测试规范描述
- 讨论中安全与规范性部分

## 2. Xiangya 安全性大样本研究（2021）
**Safety of Exercise Testing in the Clinical Chinese Population**  
用途：
- 说明临床中国人群中 exercise testing / CPET 整体是安全可规范开展的
- 支撑“算法严谨 ≠ 测试本身不安全”这层背景

建议放置：
- 引言背景
- 讨论局限与临床意义之间的过渡

# 二、中国参考值/参考方程（方法和讨论必引）

## 3. X-ET project（2021）
**Characteristics and reference values for cardiopulmonary exercise testing in the adult Chinese population – the X-ET project**  
用途：
- 中国成人 CPET 参考值的重要来源
- 支撑“必须做中国本土化校准”

建议放置：
- 引言“外部参考不一定适用中国人群”
- 方法中 external anchor 段落

## 4. Wang et al., J Clin Med 2022
**Normal References of Peak Oxygen Uptake for Cardiorespiratory Fitness Measured with Cardiopulmonary Exercise Testing in Chinese Adults**  
用途：
- 中国社区成人 cycle CPET 正常参考值
- 2nd/98th percentile 的思想很适合你当前“分位参考”路线

建议放置：
- 方法中 VO2peak 外部参考
- 讨论中“为什么 percentile 比单纯均值更适合”

## 5. Huang et al., JCRP 2024
**Reference Standards of Cardiorespiratory Fitness Measured With the Cardiopulmonary Exercise Test Using the Treadmill in Chinese Adults Younger Than 60 Years**  
用途：
- treadmill 子集的中国参考
- 用来说明 protocol mode 不能混用

建议放置：
- 方法中 mode-specific reference
- 讨论中解释 cycle / treadmill 需分开建模

## 6. PUTH equation（2025）
**A reference equation for peak oxygen uptake for cycle ergometry in Chinese adult participants**  
用途：
- 最新中国成人 cycle 参考方程
- 样本大、时间外验证明确、可直接支持“外锚点优先用中国新方程”

建议放置：
- 方法中 external anchor 主文引用
- 讨论中解释为何不再以 Wasserman/Koch 为主锚点

# 四、CPET 复合风险评分（用于反衬你为何不做简单三分类 ML）

## 9. CPX score（2008）
**A cardiopulmonary exercise testing score for predicting outcomes in patients with heart failure**  
用途：
- 说明成熟 CPET 风险模型通常是少量核心指标的复合，而不是随意多轴均值

建议放置：
- 引言末段
- 讨论中替代框架比较

## 10. CPX score validation（2013）
**Validation of a cardiopulmonary exercise test score in heart failure**  
用途：
- 支撑“复合模型 + 外部验证”的思路
- 也可说明 C-index ~0.70 左右是在疾病明确且真实结局条件下取得

建议放置：
- 讨论中“为什么 summary-level + proxy outcome 不应追同样水平”

## 11. MECKI international validation（2023）
**International validation of the Metabolic Exercise test data combined with Cardiac and Kidney Indexes (MECKI) score in heart failure**  
用途：
- 说明真正成熟的 CPET 风险模型往往需要 CPET + clinical + renal 等复合信息
- 反衬你当前 Stage I 的变量边界

建议放置：
- 讨论局限
- 未来方向

## 12. Prognostic utility / meta-analysis 文献
可补：
- HF 中 VE/VCO2 slope, peak VO2, OUES 的预后价值
用途：
- 支撑“Reserve 与 Ventilatory burden”两个主体域确有预后学基础

# 八、你在论文中最需要避免的引用误区

## 误区 1
继续大量引用欧美老参考方程，却不承认中国参考文献已更新。

## 误区 2
只引用支持 R/T/I 的一般 CPET 价值文献，却没有引用 prediction-model 方法学规范。

## 误区 3
只讲 discrimination，不讲 calibration、uncertainty 和 applicability。

## 误区 4
把 Mahalanobis、concordance、outcome-anchor 都写成并列主框架，而不说明主次。

# 九、最建议的写法

不要写：
“基于文献阈值和机器学习，我们构建了安全区预测模型。”

建议写：
“基于中国外部参考与本地 reference subset 的双锚点校准，我们构建了一个以表型负担为主体、以测试中不稳定信号为覆盖、并显式输出不确定性的实验室原型，并用 `test_result` 进行外部锚定验证。”
