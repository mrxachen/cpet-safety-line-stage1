# ADR-004: Three-Axis Anchor Framework (R/T/I)

**Status**: Accepted
**Date**: 2026-04

## Decision
Organize anchor variables into three mechanistic axes:
- R (Reserve): capacity relative to predicted
- T (Threshold): safe HR/load boundaries
- I (Instability): pathological response markers

## Rationale
Aligns with clinical exercise physiology framework; enables modular bridge design.
