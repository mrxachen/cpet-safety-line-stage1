# Bridge Sampling Priority List v1

Stage II recruitment priority for anchor validation cohort.

## Priority Tier 1 (n=60, critical)
- Age 65-80
- Hypertension + NYHA II-III
- CPET within past 12 months

## Priority Tier 2 (n=40, important)
- Age 60-75
- CAD history

## Priority Tier 3 (n=20, supplementary)
- Healthy controls (no HTN/CAD)
- Age 60-75
- Reference calibration only
