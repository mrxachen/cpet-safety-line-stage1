## 当前核心问题清单

请重点围绕以下问题展开评估：

### 问题 1：P1 循环依赖（最根本问题）
**现状**：
- P1（三区安全标签）由三个 CPET 变量确定性定义：
  ```
  vo2_peak_pct_pred < 50%  → Red
  ve_vco2_slope > 40       → Red
  eih_status == True       → Red
  ```
- `leakage_guard` 将这 3 个变量从 ML 特征集中排除（避免直接泄漏）
- **结果**：P1 标签在剩余特征空间中几乎无法预测（F1 = 0.47，与基线相当）

**核心问题**：
- 这是"标签定义即结局"的循环依赖，还是可以通过更好的特征工程解决？
- 如何设计标签，使其在汇总级 CPET 数据中具有"可学习性"？
- 是否应该完全放弃基于 CPET 参数的确定性标签，转向临床结局（如运动中不良事件、6个月心血管事件）？

### 问题 2：R/T/I 三轴等权框架的合理性
**现状**：
- Safety Score = (1/3)·VO₂轴分数 + (1/3)·通气轴分数 + (1/3)·血压轴分数
- 等权设计基于"方法学简洁性"，未经临床文献验证
- Bootstrap 95% CI 均 < 1.0（区间内不跨 Green/Yellow 边界）
- 文献 Yellow/Red 参考方案与本方法偏差 +21%

**核心问题**：
- 等权是否合理？有无 CPET 文献支持特定轴的加权？
- 三轴能否独立测量"安全"的不同维度，还是存在多重共线性？
- 是否有更成熟的多维评分方法（如 Weber 分级、CPET 复合预后评分）可借鉴？

### 问题 3：参考方程残差大（R² = 0.392）
**现状**：
- VO₂peak 参考方程（OLS，基于性别/年龄/BMI/运动方式）R² 仅 0.392
- 即使加入心血管病史，改善有限（0.298 → 0.392）
- 残差大意味着个体差异难以用人口学变量解释

**核心问题**：
- 汇总级数据中，还有哪些变量可能提升 R²？
- 是否应改用混合效应模型（随机截距捕捉个体差异）？
- 是否应放弃参考方程，直接使用百分位数参考（如 Fitness Registry FRIEND 数据库）？

### 问题 4：Phase G 三种互补方法均未达预期
**现状**：
| 方法 | 结果 | 预期 |
|---|---|---|
| Method 1 结局锚定（test_result） | AUC = 0.548，CV AUC = 0.578±0.031 | > 0.70 |
| Method 2 Mahalanobis 异常评分 | D² vs test_result r = -0.011（几乎无相关） | r > 0.30 |
| Method 3 一致性框架 | 高信度仅 26.2%（841/3206） | > 60% |

**核心问题**：
- AUC = 0.548 是否意味着当前 CPET 汇总特征根本不足以预测 test_result？
- Mahalanobis D² 与临床结局无相关，说明什么？参考子集定义有问题？
- 一致性框架中 73.5% 不确定，是标签定义本身的信噪比问题，还是特征维度不足？

### 问题 5：汇总级数据的根本瓶颈
**现状**：
- 所有 ML 方法均未超越 2 变量基线（HTN + BMI，AUC = 0.611）
- 逐呼吸时序数据尚未获取（Stage II 目标）

**核心问题**：
- 在仅有汇总级 CPET 数据的条件下，是否存在方法学天花板？
- Stage I 的最优目标应该是"精确预测"还是"框架建立 + 方向验证"？
- 如何在论文中诚实呈现这一局限，同时保持研究的创新价值？
