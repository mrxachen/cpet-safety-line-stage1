\subsection*{S1.2 条件分位参考模型技术细节}
\addcontentsline{toc}{subsection}{S1.2 条件分位参考模型技术细节}

\textbf{模型架构}：采用线性分位数回归（pinball loss 最小化）。
特征工程：自然三次样条基函数（$K$=5个内部节点，度数3）处理年龄的连续非线性效应；
哑变量（独热编码）分别处理性别（男/女）和运动方案（踏车/平板），
未知类别采用零编码处理。正则化参数设置以防止小样本严格参考子集的过拟合。

\textbf{单调性约束}：对每个样本，五个分位数预测值经排序处理，
保证 $\hat{Q}_{0.10} \leq \hat{Q}_{0.25} \leq \hat{Q}_{0.50}
\leq \hat{Q}_{0.75} \leq \hat{Q}_{0.90}$ 严格成立。

\textbf{分位模型评估}：在宽松参考子集上验证分位数校准，
检验 q10 实际覆盖率 $\approx$10\%、q90 实际覆盖率 $\approx$90\%。

\textbf{目标变量}：6个核心表型变量分别独立拟合分位模型，
各变量的分位参考束分别保存，在全量数据上独立预测个性化参考值。

\subsection*{S1.3 表型负担评分技术细节}
\addcontentsline{toc}{subsection}{S1.3 表型负担评分技术细节}

\textbf{变量级负担转换规则}：详细定义参见正文表~2（表型变量负担转换规则）。
各变量方向（越高越好/越低越好）、三级负担切点（q10/q25/q75/q90）及缺失处理规则均已在正文方法节完整列出，
此处不再重复。以下补充内容聚焦于实现细节：

\textbf{缺失值处理}：字段缺失时该变量负担记 NaN，
域内均值以可用字段平均（忽略缺失）；
若 Reserve 域可用字段 $<$2 或 Ventilatory 域可用字段 $<$1，
则对应域评分记缺失，综合表型负担 $P_{\text{lab}}$ 可能为缺失。

\textbf{表型分层切点}：
以严格参考子集上 $P_{\text{lab}}$ 分布的 P75 和 P90 为切点，
两点均由参考子集自动估计；若切点退化（P75=P90），
则 P90 替换为 P75+$\epsilon$（0.001）以避免空区间。

\subsection*{S2.1 结局锚点验证模型完整细节}
\addcontentsline{toc}{subsection}{S2.1 结局锚点验证模型完整细节}

结局锚点验证模型（ElasticNet Logistic，solver=SAGA，L1-ratio=0.5）
以全量 CPET 变量（排除验证变量）预测临床结局替代指标二元结局，
采用嵌套5折交叉验证。

\textbf{模型性能}：
交叉验证 AUC 0.564（$\pm$0.016）；测试集 AUC 0.608；
AUPRC 0.203；Brier Score 0.125；校准良好（截距约0，斜率约1）。

\textbf{定位说明}：
结局锚点验证器不定义分层边界，仅提供结局风险三分位输入置信度引擎，
并验证最终安全分层的结构效度方向。
AUC 处于 CPET 汇总指标信息量合理范围（0.55--0.65）内，
证实最终安全分层与临床结局替代指标方向一致。
LightGBM 敏感性分析结果方向与 ElasticNet 一致（均在 0.55--0.65 范围内）。

\subsection*{S3.1 置信度权重替代方案}
\addcontentsline{toc}{subsection}{S3.1 置信度权重替代方案}

当前置信度权重（0.25/0.20/0.25/0.30；完整度/努力/锚点一致/结局一致）
基于方法学先验。以下替代权重方案用于稳健性检验：

\begin{itemize}
  \item \textbf{等权方案}：四分量等权（0.25/0.25/0.25/0.25）；
    高置信度比例预计偏低（结局一致性权重降低）。
  \item \textbf{完整度优先方案}：完整度权重提升至0.60，其余等比分配；
    高置信度比例在字段完整队列中预计升高。
  \item \textbf{无结局分量方案}：结局一致性权重设为0，其余三分量等比重新分配；
    适用于无临床结局替代指标的独立外部队列。
\end{itemize}

置信度阈值敏感性（SA-3）：高置信度比例对阈值敏感
（阈值0.70时71.4\%，0.75时39.3\%，0.80时33.0\%，0.85时31.5\%），
但梯度方向在所有阈值变体下保持正向单调，方法学质量标准稳健。

\subsection*{S4.2 机器学习辅助验证结果}
\addcontentsline{toc}{subsection}{S4.2 机器学习辅助验证结果}

以下结果来自前期方法论验证，保留供方法学对照参考，
不参与本框架主要结论。

\textbf{运动前风险预测（P0）}：
XGBoost（无峰值血压特征）AUC=0.583，交叉验证 AUC=0.623$\pm$0.028；
两变量临床基线（HTN+BMI）AUC=0.611 $>$ 机器学习全特征（AUC=0.583），
反映 CPET 汇总指标的预测性能上限。

\textbf{运动后安全区分类（P1）}：
CatBoost F1\textsubscript{macro}=0.499，$\kappa$=0.302；
低危召回67.1\%，中危召回51.8\%，高危召回30.8\%。
代理泄漏消融分析：全特征 vs 无氧峰值特征 $\Delta$F1=+0.003（可忽略），
证实预测性能上限来自特征集固有局限，而非泄漏驱动。

\textbf{结局锚定模型（前期方法）}：
交叉验证 AUC=0.578$\pm$0.031，测试集 AUC=0.548；
高危区阳性率（25.3\%）$>$ 中危区（10.5\%）$>$ 低危区（5.1\%）呈单调梯度，
在本框架结局锚点验证中得到复现，验证了最终安全分层对临床结局替代指标的结构效度方向。

\subsection*{S4.3 Mahalanobis 异常表型审计详细结果}

\subsection*{S5.2 分层稳健性}
\addcontentsline{toc}{subsection}{S5.2 分层稳健性}

按以下维度分层的方法学质量标准一致性：
\begin{itemize}
  \item 性别（男/女）
  \item 年龄（$<$67岁 / $\geq$67岁）
  \item 方案（踏车 / 平板）
  \item 血压字段完整性（bp完整 / 缺失）
  \item 无血压信息敏感性（排除峰值血压相关覆盖规则）
\end{itemize}

分层验证结果：
最终安全分层（全量样本，N=3,232）及高置信度子集（N=1,065）
均呈正向单调梯度（满足预设标准）。
值得注意的是，单独使用表型负担分层时梯度方向发生反转，
说明不稳定性覆盖规则是实现结构效度的必要机制。
各分层中参考子集低危$>$50\%、高危$<$15\%的方法学质量标准均满足。

\subsection*{S5.3 多定义一致性分析}
\addcontentsline{toc}{subsection}{S5.3 多定义一致性分析}

多定义投票分析（K=4种分层定义）：
高信度841例（26.2\%），不确定区2,391例（74.0\%，含33例分类异常样本），
低危/高危冲突1,974例（61.6\%）；全样本N=3,232。
高信度高危阳性率（28.6\%）$>$ 低危（5.1\%），
证实高信度分层具有临床区分度。

本框架的置信度引擎在架构上整合了多定义一致性框架的核心精神：
以四分量加权置信度（字段完整度、努力程度、锚点一致、结局一致）
替代 K 定义投票，提供连续置信度分数而非二元高/低信度标签，
从而实现更精细的不确定性量化。
