\subsection*{S1. 机器学习辅助验证详细方法}
\addcontentsline{toc}{subsection}{S1. 机器学习辅助验证详细方法}

本研究设置两个辅助预测任务，作为方法学验证工具，旨在评估汇总级数据的预测性能上限，
而非作为核心安全区构建方法。

\textbf{运动前风险预测（Pre-exercise Risk，P0）}：
以运动前临床信息预测不良 CPET 事件风险（二分类）。
满足以下任一条件为 P0 阳性：（1）EIH 状态为阳性；
（2）VO$_2$peak $<50\%$预测值；（3）峰值SBP $>220$ mmHg。
模型采用 LASSO（5折CV $\alpha$ 搜索）和 XGBoost（100棵树，学习率0.1）；
评价指标：AUC-ROC（5折CV）。
排除峰值血压字段（含"BP"的特征），以避免用于峰值 SBP 标准的变量直接作为特征。

\textbf{运动后安全区分类（Post-exercise Safety，P1）}：
以 CPET 汇总指标预测安全区三分类（Green/Yellow/Red，有序）。
P1 标签采用 take-worst 策略，综合以下条件：
（a）EIH 状态（阳性→Red）；
（b）VO$_2$peak\%pred$<$50\%（→Red）；
（c）$V_E/V_{\text{CO}_2}$ slope$\geq$36（→Yellow 或更差）；
（d）峰值 SBP$>$220 mmHg（→Red）。
模型采用有序 Logistic 回归、LightGBM（基于梯度提升决策树）
和 CatBoost（对类别特征优化的梯度提升）；
评价指标：F1$_\text{macro}$和 Cohen's $\kappa$（5折分层CV）。

\textbf{数据泄漏防护机制（Leakage Guard）}：
由于 EIH 状态、$V_E/V_{\text{CO}_2}$ slope 和 VO$_2$peak\%pred
直接参与 P1 标签构建（确定性推导关系），
本研究设立显式数据泄漏防护机制，在 P1 特征工程阶段强制排除上述字段。
此机制确保模型性能评估的真实性：泄漏版本 F1=0.84 vs 清洁版本 F1=0.47，
差值约0.37 F1 点。
P0 任务（结局由多条件OR定义，非单一变量确定性推导）采用宽松排除策略，
仅排除直接构成 P0 条件的 SBP 相关字段。

\textbf{代价敏感分析}：
在安全优先导向下（Red 患者错误归入 Green 代价最高），
对 P1 最优模型（CatBoost）进行代价矩阵敏感性分析，
评估不同代价权重下 F1$_\text{macro}$ 与 Red 召回率的权衡。

\subsection*{S2. 结局锚定安全区模型详细方法}
\addcontentsline{toc}{subsection}{S2. 结局锚定安全区模型详细方法}

传统 P1 标签由 \{VO$_2$peak\%pred, $V_E/V_{\text{CO}_2}$ slope, EIH 状态\}
确定性定义，导致预测这些变量的 ML 模型无法突破 P1 的循环依赖，
预测性能受制于标签定义本身。

本研究提出以 CPET 临床综合结局（test\_result，阳性/可疑阳性 vs 阴性）
为标签的结局锚定方法：综合结局由临床医生对整个 CPET 检查的综合判断决定，
不由任何单一 CPET 变量确定性推导，因此使用全部 CPET 指标作为特征不构成泄漏。

\textbf{模型参数}：
LightGBM 二分类器，全 CPET 特征集（含 VO$_2$peak \%pred、
$V_E/V_{\text{CO}_2}$ slope、EIH状态等11个特征），
5折分层交叉验证超参数搜索（RandomizedSearchCV，n\_iter=40，
搜索空间：num\_leaves 20--127，learning\_rate 0.01--0.3，
n\_estimators 100--500），
isotonic 回归校准（CalibratedClassifierCV，cv=2）。

\textbf{切点确定}：
安全区切点在训练集上确定以防止测试集信息泄漏：
Green/Yellow 界采用敏感度约束法（敏感度$\geq$90\%的最高阈值）；
Yellow/Red 界采用 Youden's J 最大化。
评价指标：AUC-ROC（5折CV）、测试集 AUC、Average Precision（AP）、Brier Score。

\subsection*{S3. 多变量 Mahalanobis 异常评分详细方法}
\addcontentsline{toc}{subsection}{S3. 多变量 Mahalanobis 异常评分详细方法}

传统\%pred 逐变量比较无法捕获 CPET 变量间的协方差信息。
本研究以参考正常子集（CTRL 组，经缺失值筛选后 n=969）
构建多变量 CPET 空间中的 Mahalanobis 距离 $D^2$，公式如下：

\begin{equation}
D^2 = (\mathbf{x} - \boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})
\end{equation}

其中 $\boldsymbol{\mu}$ 和 $\boldsymbol{\Sigma}$ 分别为参考子集的均值向量和协方差矩阵。
在多变量正态参考人群中，$D^2 \sim \chi^2(k)$（$k$=变量数），
提供概率解释：$D^2 > \chi^2(k, 0.95)$ 意味着患者处于参考人群95百分位以上。

\textbf{5个核心变量}：VO$_2$peak、HR$_\text{peak}$、O$_2$脉搏峰值、OUES、峰值METs。
\textbf{切点方法}：参考子集经验分位法（P75/P95），
以参考子集 $D^2$ 分布的经验百分位作为 Green/Yellow 和 Yellow/Red 分界，
与理论 $\chi^2$ 分位数互相验证。

\subsection*{S4. 多定义一致性框架详细方法}
\addcontentsline{toc}{subsection}{S4. 多定义一致性框架详细方法}

本研究提出以分歧本身作为信息的一致性框架：
利用 K=4 个独立安全区定义对每个患者进行多数投票，
将定义间的系统性分歧量化为患者级别的分类不确定性。

\textbf{K=4 定义}：
\begin{enumerate}
    \item R/T/I 数据驱动 Zone（本研究主框架）
    \item 结局锚定 Zone（基于 CPET 综合结局的 LightGBM 切点）
    \item Mahalanobis 异常 Zone（基于 $D^2$ 经验切点）
    \item VO$_2$peak 简单阈值 Zone（仅基于 VO$_2$peak\%pred 的单变量分层）
\end{enumerate}

\textbf{一致性判定规则}：
\begin{itemize}
    \item \textbf{高信度区（一致区）}：最高票数/K $\geq$ 0.60 且无 Green/Red 同时出现
    \item \textbf{不确定区}：Green/Red 同时出现（高危冲突）或无多数票（最高票数/K $\leq$ 0.50）
\end{itemize}

区域置信度（最高票数/K）提供患者级别的分类可靠性量化。
不确定区患者代表各定义存在实质分歧的病例，
是需要更精细波形级评估的患者亚群。
